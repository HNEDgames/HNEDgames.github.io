# 엔드필드 계산 사이트

## 전력 계산

### 무릉 배터리 분할

```math
무릉\ 배터리\ 1개 = 40초동안\ 1600W\ 생성
```

```math
무릉\ 배터리\ 1.5개 = 1분동안\ 1600W\ 생성
```

```math
출력포트 = 1분에\ 30개\ 출력 = 1600W \times\frac{ 30}{1.5} = 32000W
```

```math
출력포트를\ 6400등분하면\ 5W
```

```math
출력\ 포트\ \frac{1}{6400} = \frac{1}{2^{8}\times 5^{2}}\ 마다\ 5W\ 생성
```

```math
따라서\ 5등분 공정\ 2개와,\ 2등분공정\ 8개를\ 통해\ 5W\ 단위로\ 조절\ 가능하다.
```
